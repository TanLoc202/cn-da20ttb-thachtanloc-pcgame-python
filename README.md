# Đề tài: Phát triển trò chơi PC với Python

**Giang viên hướng dân: Trần Văn Nam**

**Sinh viên thực hiện: Thạch Tấn Lộc**

## Mô tả đề tài

Đề tài tập trung vào nghiên cứu và ứng dụng ngôn ngữ lập trình Python kết hợp với thư viện pygame, để xây dựng một trò chơi 2D hoàn trỉnh chạy trên nền tẳng PC. Bằng cách vận dụng các kiến thức về lập trình hướng đối tượng (OOP) để thiết kế kiến trúc game, xử lý các yếu tố cốt lõi như vòng lập game (game loop), xử lý sự kiện đầu vào, cơ chế vật lý cơ bản, phát hiện va chạm, quản lý trạng thái,... Đồng thời tích hợp các tài nguyên như đồ họa, âm thanh,...

## Cài đặt môi trường

### Cài đặt Python

Phiên bản pỵthon được dung trong đề tài này là Python 3.13.6. Link download: [Python Release Python 3.13.6 | Python.org](https://www.python.org/downloads/release/python-3136/)

### Cài đặt thư viện Pygame bằng pip

```bash
pip install pygame==2.6.1
```
